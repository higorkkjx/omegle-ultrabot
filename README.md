# omegle-ultrabot
Bot omegle para android e pc, resolvendo recaptcha automaticamente e identificando se você caiu na blacklist do site.

### O que precisa:

- NodeJS
- Termux (Se for usar no celular)

### Instalação:

- git clone https://github.com/higorkkjx/omegle-ultrabot
- cd omegle-ultrabot
- node .
